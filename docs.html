# about

temPHPlate is a template structured in MVC with simplified syntax and easily to use for high-level project development.

this template provides a structure and starting point for creating your application, allowing you to focus on creating something amazing while we sweat the details.

temPHPlate strives to provide a great developer experience, while providing various features such as thorough dependency injection, an expressive database abstraction layer, a view manipulation system, and more.

# installation

installation via composer

if your computer already has PHP and Composer installed, you may create a new temPHPlate project by using Composer directly.

            
              $ composer create-project otaviothor/temphplate your-project-name

installation via github

Or, you can start a temPHPlate project cloning from github, just have the git CLI installed.

          
            $ git clone https://github.com/otaviothor/temphplate your-project-name
